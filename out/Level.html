Class: Level

Level(enemies, endboss, clouds, coins, backgroundObjects, bottleInSand)

The Level class represents a game level, including its enemies, endboss, clouds, coins, background objects, and bottles in sand.

Constructor

new Level(enemies, endboss, clouds, coins, backgroundObjects, bottleInSand)

Constructs a new Level instance.
Parameters:
Name Type Description

endboss Object The endboss object of the level.
clouds Array The array of cloud objects in the level.
coins Array The array of coin objects in the level.
backgroundObjects Array The array of background objects in the level.
bottleInSand Array The array of bottle objects in the sand in the level.
